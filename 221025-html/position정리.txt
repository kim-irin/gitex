static : 기본값. 리셋값

relative : 내가 있는 기준으로 방향값만큼 움직여서 고정되어있다

absolute : 층위로 올라가 부모요소의 값을 기준으로 방향만큼 움직여 고정되어있다
             부모가 없으면 바디값부터 시작

fixed : 브라우저를 기준으로 층위로 올라와서 스크롤을 내려도 화면에 붙어있다

sticky : 내가 정한 margin값을 스크롤이 지나면 화면에 고정
	(%,overflow:hidden,방향없음)	이면 못씀

z-index : 위치조절. 숫자가 클수록 위로감
